本地参数查到了，并且我用 `psql` 实测连接成功了。你截图里失败是因为用户名填成了 `sub2api`，本地实际是 `postgres`。

安装向导里填：

```text
主机: 127.0.0.1
端口: 5432
用户名: postgres
密码: postgres
数据库名称: sub2api
SSL 模式: 禁用
```

本地配置来源是 [backend/config.yaml](E:/2.project/sub2api/backend/config.yaml)：

```yaml
database:
  host: 127.0.0.1
  port: 5432
  user: postgres
  password: postgres
  dbname: sub2api
  sslmode: disable
```

我验证结果：

```text
current_user: postgres
current_database: sub2api
PostgreSQL: 15.18
```


Redis 本地参数就是：

```text
主机: 127.0.0.1
端口: 6379
密码: 留空
数据库: 0
启用 TLS: 关闭
```

我已经帮你把本地 Redis 启起来了，验证结果是：

```text
PONG
127.0.0.1:6379 正在监听
```

安装向导 Redis 那一步按上面填即可。
